Trillium | Validator Leaderboard Guide

Introduction

Welcome to the Trillium Validator Leaderboard! This dashboard is a powerful tool designed to provide comprehensive insights into validator performance on the Solana network. This guide will walk you through its features to help you make the most of the available data.

Getting Started

The two main controls at the top of the page are:

  • Epoch Selector: Allows you to view historical performance data for a specific epoch.
  • Metric Group Selector: Organizes validator metrics into logical groups (e.g., Basic Information, Vote Performance, APY Metrics). You can switch between these groups to explore different facets of validator data.

Key Features

Persistent Selections & Filters

One of the most powerful features of this leaderboard is its persistence. Any filters you apply, validators you select for comparison, or the epoch you are viewing will remain active even when you switch between different Metric Groups.

This is achieved by storing your selections in the URL. This allows you to analyze a consistent set of validators across various performance categories without having to re-apply your filters. You can also share the URL with others to show them the exact same data you are seeing.

Epoch-over-Epoch Performance Changes

To help you quickly identify trends, the table indicates how a validator's metrics have changed compared to the previous epoch.

  • A green up arrow ▲ next to a value signifies an improvement in that metric (e.g., higher APY, more blocks produced). For rank-based metrics, an up arrow means a better (lower) rank number.
  • A red down arrow ▼ signifies a decline in that metric (e.g., lower APY, higher skip rate).

Hovering your mouse over an arrow will reveal a tooltip showing the precise percentage change from the previous epoch, giving you a deeper understanding of the magnitude of the change.

Filtering and Sorting

Each column in the table can be sorted by clicking on its header. You can also apply fine-grained filters:

  • Click the filter icon in the column header to open a filter popup.
  • You can filter by name, public key, client type, version, and a range of values for numerical metrics.
  • The "Averages for Filtered Validators" tab provides aggregate statistics for the set of validators that match your current filters.

Validator Comparison Chart

You can visually compare the performance of up to 15 validators:

  1. Use the checkboxes in the "Select" column to choose the validators you want to compare.
  2. Once you have selected at least one validator, click on the "Chart" tab.
  3. The chart will display a visual comparison of the selected validators for the metrics in the currently active group (available for Performance, Averages, and Slot Durations groups).
